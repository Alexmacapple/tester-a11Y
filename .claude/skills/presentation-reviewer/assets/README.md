# Assets - Presentation Reviewer

Ce dossier contient les fichiers assets utilisés par le skill presentation-reviewer.

## Contenu

### template-dsfr.pptx
Template PowerPoint conforme au Design Système de l'État Français (DSFR).

**Caractéristiques:**
- Format 16:9
- Logos République Française et DGAFP
- Palette de couleurs DSFR
- Typographie Marianne (ou Arial en fallback)
- Mise en page conforme aux standards d'accessibilité RGAA

**Usage:**
- Référence pour analyser la conformité DSFR
- Template de base pour nouvelles présentations
- Exemple de bonnes pratiques DSFR

## Comment utiliser ces assets

Les assets ne sont pas chargés dans le contexte de Claude mais sont utilisés comme:
- Templates à copier et personnaliser
- Exemples de référence pour l'analyse
- Modèles à reproduire lors de corrections
